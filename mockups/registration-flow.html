What changes (in plain terms)

  1. The old "Choose your role" screen is removed. Role is now picked inside the new profile form.
  2. After OTP, a new user fills Name, Date of birth, Address, and Role on one screen and taps Submit for approval.
  3. The submission creates a pending registration request visible to the admin (table above).
  4. The user is blocked from logging in — every login attempt lands on "Awaiting admin review" until a decision is made.
  5. Admin Accept → user becomes active and can log in. Reject → denied (wording TBD).
  6. Every new string ships in all 4 languages (English, العربية, کوردی سۆرانی, کوردی بادینی) with correct RTL.

👉 Switch languages with the buttons up top, pick a role in screen 3, and click Accept in the admin table to watch the phone flip to "approved".

⚠️ Badini (Kurmancî) strings are a baseline translation — same caveat the app's kmr locale file already notes; a fluent reviewer should confirm before shipping.
